## Rocket UniVerse Release Notes 

These release notes provide information about the customer issues and enhancements made for UniVerse. 

## Important news at this release 

- UniVerse 11.2.4 now ships with the OpenSSL 1.0.1h library, eliminating the Heartbleed vulnerability. UniVerse 11.2.3 incorporated an interim fix to this vulnerability as described in UNV-18620. 

- When using the General Call Interface (GCI) at UniVerse 11.2.4 on UNIX/Linux platforms, a C++ compiler is required to build a new uvsh binary. Do **not** upgrade to 11.2.4 if you are using GCI and do not have a C++ compiler available. 

   - UniVerse 11.2.x and higher use third party XML Libraries that are compiled using C++.  To accommodate this, starting at UniVerse 11.2.4, the linking executable used in the GCI make files was changed from the C to the C++ compiler for each platform.
