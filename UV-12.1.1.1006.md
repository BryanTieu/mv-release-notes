## Rocket UniVerse overview 

These release notes provide information about the customer issues and enhancements made for UniVerse. 

## Important news at this release 

The Recoverable File System (RFS) has been added to UniVerse. If RFS_MODE is set to 1 in `uvconfig` , then extreme caution should be used when manipulating files at the OS level. 

Refer to the RFS documentation for additional information. 

**Warning:** The `uvdb` user is required. Permissions on files are now owned by the uvdb user.
